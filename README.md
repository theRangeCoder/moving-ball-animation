<!-- GETTING STARTED -->
## Getting Started

To get a local copy of the animation, up and running on your system, follow these steps.


### Installation

1. Fork this repository
<div style="text-align:center"><img src="./images/fork.png" /></div>

2. Clone the repository
   ```sh
   git clone https://github.com/[your_username]/moving-ball-animation.git
   ```
3. Navigate into the directory
   ```sh
   cd moving-ball-animation
   ```
4. Open the `index.html` file on your default browser or a browser of your choice
   ```sh
   xdg-open ./index.html
   ```
